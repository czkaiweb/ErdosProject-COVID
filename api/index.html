This forecasting tool uses machine learning algorithms to predict the number of hospital beds filled by COVID-19 inpatients in each state, up to 14 days in the future. To learn more about our methods or access the source code, visit our GitHub repository.
You can also try our API. From the command line, enter e.g.
curl covidforecast.net:5000/?state=AL

Created by Lara Du, Julian Rosen, and Kai Wei as a project for the Erdős Institute's Fall 2021 data science mini-course. Our forecast is intended for educational purposes only, we cannot guarantee its accuracy.
